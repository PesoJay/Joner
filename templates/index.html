Joner - Practice Sightreading

Joner is a Web App designed to help you practice Sightreading in a way that best suits your needs. Wether you need to practice playing chords, want to dive into randomly generated pieces, or if you just want to practice hitting single notes on sight: Joner has you covered! If you want to learn more about each mode, just hover over the button and read the tutorial. Before starting a Full Read Practice Session be sure to do the latency test, for better accuracy.
